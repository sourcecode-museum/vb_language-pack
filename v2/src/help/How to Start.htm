VB Language Pack Generator
 

1. How to Start Implementing

To add Language Pack support in your Visual Basic Project it's not needed to create a Language Pack first, but you can create it, it doesn't affect the way you will implement, so i'ts your decision. As you can see, a Class Module is in the "Use of LanguagePack" folder called "clsLanguagePack.cls". Add the Class Module to your Visual Basic Project then create a Module or use an existent if you have one to declare the Language Pack Object Variable.

' Use this code to declare the Object in a Module
Global cLanguage As New clsLanguagePack
