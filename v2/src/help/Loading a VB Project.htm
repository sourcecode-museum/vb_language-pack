VB Language Pack Generator
 

1. Loading a VB Project

To start creating your first Language Pack you need to load a VB Project first.
Just Start VB Language Pack Generator and click the "Load VBP" button then select a Visual Basic Project.

You will see that all forms of your project are listed in the "Project list" at the left, like in Visual Basic.
You can click in a form, so a list of controls will appear in the "Object list" at the right, you can click in the object name to see its caption and tooltiptext.
