VB Language Pack Generator
 

1. Using Strings

The code below shows how to use strings that you created with the Language Pack:

' It shows a MsgBox with the string "Test" if it exists and is set
' If the string doesn't exists or isn't set the app shows "Test!" as default.
MsgBox IIf(cLanguage.GetString("Test") <> "", cLanguage.GetString("Test"), "Test!")
