VB Language Pack Generator
 

1. Loading a Language Pack

The code below shows how to load a Language Pack selected in a ListBox and in the subfolder "packs".

' Just to be sure if there is Language Packs loaded or if the user selected one
If ListBox.ListCount = 0 Or ListBox.ListIndex = -1 Then Exit Sub

' Loads the entire language pack. It doesn't apply the language pack in the form.
cLanguage.LoadPack App.Path & "\packs\" & ListBox.List(ListBox.ListIndex)
