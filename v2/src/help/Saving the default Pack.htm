VB Language Pack Generator
 

1. Saving the "default" Pack

To save a "default" Language Pack of your project, just generate it without translating any controls. Just load your Visual Basic Project, select an "Output File" then click "Generate Pack". It will create a Pack with all default values of captions and tooltips.
