1. Enumerating Language Packs

The code below shows how to fill a listbox with all Language Packs in the subfolder "pack" of the Project.

' Clear the listbox
ListBox.Clear

' Temp variables
Dim sTmp As String, sTmpArray() As String, i As Integer

' Set the temp var with the function that returns the packs found separated by |
sTmp = cLanguage.EnumLanguagePacks(App.Path & "\packs", "*.lpk")
' Lets split the temp variable into the temp array
sTmpArray = Split(sTmp, "|")
' Lets put the files into the listbox
For i = 0 To UBound(sTmpArray)
  ' Just to be sure that it's not empty
  If sTmpArray(i) <> "" Then ListBox.AddItem sTmpArray(i)
Next
