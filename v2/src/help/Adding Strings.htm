VB Language Pack Generator
 

1. Adding Strings

Everytime you load a Visual Basic Project or an existing Language Pack the "Project list" will load all forms of your project and the last item will always be "Strings". To add a string you need to click in the "Strings" item then click in "Add String" in the "Object list" at the right. Set the name of the string and the string value of it. You can edit strings later just clicking on the name of the string. Easy!
