1. Translating Controls

Translate controls of your project is a very easy task.
Just load your Visual Basic Project, then select a form, so you will see the "Object list" at the right. Just click in a control then type the translated caption in the "New Caption" textbox and the translated tooltip in the "New Tooltip" textbox. Leave it blank if it doesn't have a default value.
